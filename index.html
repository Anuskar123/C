üöÄ Quick Reference Cheat Sheet

Essential C++ syntax & concepts at your fingertips

üìö Basic Syntax

üî• Program Structure:

#include <iostream>
#include <string>
using namespace std;
int main() {
    // Your code here
    return 0;
}

üî• Variable Declarations:

int age = 25;              // Integer
double price = 19.99;      // Decimal number
char grade = 'A';          // Single character
string name = "John";      // String of characters
bool isTrue = true;        // Boolean (true/false)

üî• Input/Output:

cout << "Hello World!";    // Output
cin >> variable;           // Input (stops at whitespace)
getline(cin, stringVar);   // Input entire line

üîÑ Control Structures

üî• IF-ELSE:

if (condition) {
    // code
} else if (condition2) {
    // code
} else {
    // code
}

üî• LOOPS:

// For loop
for (int i = 0; i < 10; i++) {
    // code
}
// While loop
while (condition) {
    // code
}
// Do-while loop
do {
    // code
} while (condition);

üéØ Functions

üî• Function Syntax:

returnType functionName(parameters) {
    // code
    return value;  // if not void
}
// Examples:
int add(int a, int b) {
    return a + b;
}
void printMessage() {
    cout << "Hello!";
}

üìä Arrays

üî• Array Basics:

int arr[5];                    // Declaration
int arr[5] = {1, 2, 3, 4, 5}; // Initialization
int arr[] = {1, 2, 3};        // Size inferred
// Accessing elements
arr[0] = 10;   // First element
arr[4] = 50;   // Last element (for size 5)

üîó Pointers

üî• Pointer Basics:

int x = 10;
int* ptr = &x;     // ptr holds address of x
cout << *ptr;      // Dereference: get value at address
*ptr = 20;         // Change value through pointer

üö® Common Mistakes

  • ‚ùå Using = instead of == in conditions
  • ‚ùå Array bounds violations (accessing arr[size])
  • ‚ùå Forgetting to initialize variables
  • ‚ùå Memory leaks (new without delete)
  • ‚ùå Missing base case in recursion
  • ‚ùå Integer division when expecting decimal
